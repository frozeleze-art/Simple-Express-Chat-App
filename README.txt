How to run simple-express-app on Windows

1. Unzip the archive.
2. Open PowerShell and run:
   .\simple-express-app.exe

By default the server listens on port 3000. To change port before running:

PowerShell:
$env:PORT=4000; .\simple-express-app.exe

To stop the server, press Ctrl+C in the window running the exe or find and kill the process.
